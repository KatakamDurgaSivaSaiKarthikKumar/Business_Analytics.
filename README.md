# 📊 Business Analytics Dashboard

A modern and interactive **Business Analytics Dashboard** developed using **HTML, CSS, and JavaScript**. This project provides a user-friendly interface for monitoring business performance, analyzing sales, managing customers, generating reports, viewing AI-based suggestions, and customizing user settings.

---

## 📌 Project Overview

The Business Analytics Dashboard is designed to help business owners visualize and manage important business information through an intuitive dashboard.

The project includes multiple modules such as:

- 🏠 Home Dashboard
- 📈 Business Overview
- 💰 Sales Analytics
- 👥 Customer Management
- 📋 Reports
- 💡 Business Feedback & Suggestions
- ⚙️ Settings
- 🔐 Login Authentication

The dashboard focuses on providing a clean UI, responsive design, and an enhanced user experience.

---

# 🚀 Features

### 🔐 Login System
- Password-based authentication
- Incorrect password validation
- Redirects to dashboard after successful login
- Password can be changed from Settings page

### 🏠 Dashboard
- Business summary cards
- Navigation sidebar
- Responsive layout
- Interactive user interface

### 📊 Overview
- Displays important business metrics
- Quick business insights
- Performance summary

### 💰 Sales
- Sales analysis
- Revenue information
- Sales statistics

### 👥 Customers
- Customer details
- Customer statistics
- Business customer management

### 📋 Reports
- Business reports
- Performance tracking
- Data presentation

### 💡 Suggestions
- Business improvement suggestions
- Recommendations for better growth

### ⚙️ Settings
- Profile Information
- Profile Picture
- Password Change
- Notification Settings
- Dark Mode
- Theme saved using Local Storage
- Responsive UI

### 🌙 Dark Mode
- One-click dark/light mode switch
- Theme preference saved using Local Storage
- Applied across all dashboard pages

### 🔔 Notification Settings
- Email Notifications
- SMS Notifications
- Push Notifications
- Toggle Switch Interface

### 📱 Responsive Design
- Desktop Friendly
- Tablet Friendly
- Mobile Friendly

# ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/KatakamDurgaSivaSaiKarthikKumar/Business-Analytics.git
```

### 2. Open the Project

Open the project folder in **Visual Studio Code**.

### 3. Run

Simply open

```
login.html
```

or use **Live Server Extension** for the best experience.
